[Floating Islands - Savanna][gitlab]
====================================

A Minecraft map featuring four savanna-themed floating islands, originally made for my [Bedwars plugin][bedwars-plugin].

I created this map as part of the [minigame section][plugin-course-minigames] of the [Udemy course][plugin-course] for developing Minecraft plugins by [Stephen][plugin-course-author].

![Savannah floating islands][project-image]

Features
--------

* 2-4 teams
* 1 island per team
* Savanna biome theme
* Acacia tree on each island
* Coal and iron ores in the islands
* Compatible with my [Bedwars plugin][bedwars-plugin]

See the [Bedwars plugin][bedwars-plugin] page for more information.
